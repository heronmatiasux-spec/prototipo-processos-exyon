# Protótipo Litigância E-XYON

Arquivos prontos para subir no GitHub Pages.

## Estrutura que deve ir para o repositório

- `index.html`
- `styles.css`
- `.nojekyll`
- `assets/`
- `processos/`

## Navegação

- A dashboard abre em `index.html`.
- O card totalizador `Processos` leva para `./processos/`.
- O botão `Voltar`, dentro da página de processos, retorna para `../`.

Esses caminhos são relativos, então funcionam tanto localmente quanto no GitHub Pages.
